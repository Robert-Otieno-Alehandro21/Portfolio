Projects

Below are some of the projects I've worked on — from full-stack systems to tailored dashboards — each crafted with purpose, precision, and impact.

RCMS - Real Construction Management System

RCMS is a comprehensive platform built to simplify the construction ecosystem. Users can create projects, assign roles, upload progress images, approve milestones, and integrate banking for payments. The system handles land selection using geolocation, real-time photo uploads, and detailed project tracking. I led the frontend architecture using React.js and implemented dynamic dashboards for each role — Users, Project Managers, Construction Companies, and Banks.

This project deepened my understanding of multi-role systems, React routing, and modular architecture. It also taught me the importance of mobile responsiveness, clean UI, and real-time feedback in complex platforms.

NaviPOS - Smart Point of Sale System

NaviPOS is a modern, lightweight POS solution for small and medium businesses. It features real-time inventory management, customer tracking, sales analytics, and receipt printing. Built with React and Node.js, it offers offline-first functionality and a clean, intuitive interface that's easy for cashiers and managers to operate.

I focused on optimizing performance, securing transactions, and integrating local data storage using IndexedDB. The goal was to offer something functional and beautiful — and the result was a sleek POS used by actual clients for daily business.

Church Management System

This system was built to help churches manage member registration, sermon schedules, announcements, event planning, and financial records. The dashboard includes modules for admin, pastors, and department heads, enabling role-specific control over content and tasks.

Designed with a spiritual yet modern aesthetic, the system allowed me to explore clean UI/UX principles while managing sensitive data structures and real-time updates in faith-based organizations.

School System (Student Results & Administration)

This system enables administrators to manage student enrollment, exam results, grading, term reports, and performance analysis. It includes secure logins for students, parents, and staff, with customized dashboards for each role. Data is stored securely with role-based access controls.

I implemented detailed reporting tools and PDF exports for results, and the experience taught me the real-world implications of data security, student privacy, and automated report generation.
