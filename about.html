My Journey at Xiracom Limited

My journey at Xiracom Limited was a defining period in my development as a software engineer. It wasn't just an internship — it was a full immersion into the pace and professionalism of a working tech team.

I joined Xiracom during my university attachment period and was assigned to work on the CMS (Church Management System), a platform that streamlines operations for a church. This project gave me practical exposure to collaborative coding and real-world deployment standards.

At Xiracom, I worked on developing role-based dashboards. Each user — whether a client, project manager, or contractor — had their own experience, and I was responsible for building those interfaces using React.js and modern CSS. I learned the importance of usability and clear design when working with real stakeholders.

I implemented modules that allowed users to upload construction photos, view project timelines, manage teams, and make approvals. I also assisted in integrating payment verification features and location selection using longitude and latitude.

One of the most important lessons I learned was how to work with clean code architecture and reusable components, ensuring that the codebase was scalable and maintainable. I got hands-on experience working with version control systems and testing APIs.

I improved my ability to work under tight timelines and learned how to communicate clearly with supervisors and team members. There were moments of debugging long into the night — and in those, I discovered the true grind and reward of being a developer.

Beyond technical skills, Xiracom taught me professionalism, time management, and how to carry myself with confidence in a software environment. I walked out of the experience more mature, more capable, and ready to take on even bigger systems.

Technical Skills

My core skills include HTML, CSS, JavaScript, and React for frontend development. I've built multiple responsive interfaces and understand how to build clean, scalable component structures.

On the backend, I'm proficient with Node.js, Express, and MongoDB. I've worked with Supabase for authentication and file storage, and I've integrated third-party services like Cloudinary for image management.

I use GitHub for version control, Postman for API testing, and deployment platforms like Render, Netlify, and Vercel. I'm comfortable working with RESTful APIs and enjoy building systems that are both secure and performant.
